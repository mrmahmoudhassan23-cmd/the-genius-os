# The Genius OS

The Genius OS is the central operating system for managing teaching, educational content, planning, documentation, branding, templates, and business operations.

## Repository Structure

- docs
- scripts
- calendar
- notion
- branding
- templates
- assets
- archive

## Project Standards

- Modular architecture across all engines and modules
- Consistent naming conventions defined in core/Naming.md
- Structured workflows defined in core/Workflows.md
- Separation of concerns between modules and areas
- Documentation-first development approach
